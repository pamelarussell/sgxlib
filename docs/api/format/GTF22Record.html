  8. lazy val end: Int

    Zero-based end position in reference coordinates, exclusive.

    Zero-based end position in reference coordinates, exclusive.

    GTF2.2 format uses 1-based positions, and furthermore, uses fully closed intervals so the end position is the last position included in the feature. This library uses 0-based positions and half open intervals, so the end position is the position after the last position included in the feature. Therefore, this value is equal to the integer in the GTF2.2 file, because it is theoretically converted twice for these two differences.

    Returns a new FeatureBuilder consisting of the passed FeatureBuilder with the information in this GTF22Record added to it.

    The feature type must not be one of the CNS types (see format specification) as these features are specified on a single GTF2.2 line and should not be incorporated into an existing FeatureBuilder. The feature type must also not be an ignored type.
